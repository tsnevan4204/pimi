§1

Review: Stokes Flow as a Saddle Point Problem

Last time we discussed locking and showed that Stokes flow is a special limit of incompressible elasticity.

$$\nabla^2 u - \nabla p = f \qquad \text{(Lagrange multiplier)}$$ $$\nabla \cdot u = 0 \qquad \text{(incompressibility constraint)}$$

This is an example of an abstract saddle point problem.

§2

Abstract Saddle Point Framework

The abstract form of the problem is: find $u \in V$, $p \in M$ such that

(★) $$a(u, v) + b(p, v) = L_1(v) \qquad \forall\, v \in V$$ $$b(\xi, u) \phantom{+ b(p, v)} = L_2(\xi) \qquad \forall\, \xi \in M$$
Remark — Matrix Form This corresponds to the block linear system $$\begin{pmatrix} A & B \\ B^T & 0 \end{pmatrix} \begin{pmatrix} \hat{u} \\ \hat{p} \end{pmatrix} = \begin{pmatrix} b_1 \\ b_2 \end{pmatrix}$$ where $A$ comes from the bilinear form $a(\cdot,\cdot)$ and $B$ from $b(\cdot,\cdot)$.
§3

The Inf-Sup (LBB) Condition

Theorem Assume $a(\cdot,\cdot)$ satisfies Lax-Milgram. Then (★) has a unique solution if $V$ and $M$ satisfy an inf-sup (aka LBB) condition.
Definition — Inf-Sup (LBB) Condition $V$ and $M$ satisfy the inf-sup condition (are inf-sup compatible) if $$\forall\, p \in M, \quad \exists\, \tilde{v} \in V \text{ nonzero s.t.}$$ $$\frac{b(p, \tilde{v})}{\|\tilde{v}\|_V} \geq \beta \|p\|_M$$ for some constant $\beta > 0$.
Proof sketch
  • Recall: $Ax = b$ has a solution; if $b = 0 \Rightarrow x = 0$.
  • Take $L_1 = L_2 = 0$, $\xi = p$, $v = u$. Then $$a(u,u) + b(p,u) = 0, \quad b(p,u) = 0 \;\Rightarrow\; a(u,u) = 0.$$
  • Coercivity: $0 = a(u,u) \geq \alpha\|u\|_V^2 \;\Rightarrow\; u = 0$.
  • By inf-sup, $\exists\, \tilde{v} \in V$. Take $v = \tilde{v}$: $b(p, \tilde{v}) = 0$, so $$0 = \frac{b(p,\tilde{v})}{\|\tilde{v}\|_V} \geq \beta\|p\|_M \;\Rightarrow\; p = 0.$$
§4

Concrete 1D Example ($\Omega = [0,1]$)

Consider $-u'' = f$. We rewrite as a first-order system by setting $F = u'$:

$$F = u' \qquad -F' = f$$

Pick $F \in V$, $u \in M$. For all $v \in V$, $\xi \in M$:

$$(F, v) - (u', v) = 0$$ $$-(F, \xi) = (f, \xi)$$

After integration by parts (IBP) on the top equation:

$$(F, v) + (u, v') = uv\big|_0^1$$ $$-(F', \xi) = (f, \xi)$$

We have a saddle point problem with bilinear forms:

$$a(F, v) = (F, v), \qquad b(u, v) = (u, v')$$

Note: $a(\cdot,\cdot)$ satisfies Lax-Milgram (it's just the $L^2$ inner product). Now let's build an inf-sup compatible space.

Construction Let $$M_h = \{f \in L^2(\Omega) : f|_e = \mathrm{const}\}, \qquad V_h = \text{piecewise linears}$$ Task: Given any $p \in M_h$, build $\tilde{v} \in V_h$ such that $\tilde{v}' = p$. Hint: Think FTC — build an antiderivative.

Set $\tilde{v}_0 = 0$ and define $\tilde{v}_j = \sum_{k=0}^{j-1} p_k h$. Then on element $e_j$:

$$\int_{e_j} \tilde{v}'\, dx = \tilde{v}_{j+1} - \tilde{v}_j = p_j h$$

This construction is building an antiderivative — easy in 1D.

Now compute $b(p, \tilde{v})$:

$$b(p, \tilde{v}) = \int p\, \tilde{v}'\, dx = \sum_j \hat{p}_j(\tilde{v}_{j+1} - \tilde{v}_j) = \sum_j \hat{p}_j^2 h = \|p\|_{L^2}^2$$

And the $H^1$ seminorm of $\tilde{v}$:

$$|\tilde{v}|_{H^1}^2 = \sum_{e_j} \int_{x_j}^{x_{j+1}} (\tilde{v}')^2\, dx = \sum_{e_j} \int_{x_j}^{x_{j+1}} p^2\, dx = \|p\|_{L^2}^2$$

Therefore:

$$\frac{b(p, \tilde{v})}{|\tilde{v}|_{H^1}} = \frac{\|p\|_{L^2}^2}{\|p\|_{L^2}^2} = \|p\|_{L^2}$$
Result Inf-sup holds with $\beta = 1$, using norms $V = H^1$ seminorm, $M = L^2$.
Thoughts on Inf-Sup
  • It comes from the FTC and building an antiderivative. Can we do this in higher dimensions?
  • If we pick a bigger space $V_{\mathrm{big}} \supseteq V$, then $V_{\mathrm{big}}$ is also inf-sup compatible. Is $V_{\mathrm{big}}$ big enough to find an antiderivative?
  • Formally, $V$ and $M$ are surjective under differentiation.
§5

Vector Calculus & Conservation Laws

We break to revisit vector calculus. Objective (until next week): steady-state dynamics.

$$Au + \varepsilon N[u] = f$$

We'd like to pose this in conservative form:

$$\nabla \cdot F = f \qquad \leftarrow \text{exact! conservative}$$ $$F = -\nabla u + N[u]$$

Then any learned model will be conservative if $\nabla \cdot \cdot = 0$:

$$0 = \int \nabla \cdot F\, dx = \oint F \cdot dA \qquad \text{(what goes in = what comes out)}$$
Other Flavors of Conservation (Generalized Stokes Theorem) $$\int \nabla \phi \cdot dl = \phi_j - \phi_i$$ $$\int \nabla\!\times\!\psi \cdot dA = \int \psi \cdot dl$$ $$\int \nabla \cdot F\, dV = \oint F \cdot dA$$

These are all instances of:

$$\int_\Sigma dF = \int_{\partial\Sigma} F \qquad \text{(Generalized Stokes Theorem)}$$
§6

The de Rham Complex & FEM Spaces

Define the differential operators: $d_0 = \nabla$, $d_1 = \nabla\times$, $d_2 = \nabla\cdot$.

Key Identity $$\nabla\!\times\!\nabla = \nabla\cdot\nabla\!\times = 0 \;\;\Longrightarrow\;\; d_{k+1}\, d_k = 0$$

We want to build families of FEM spaces that form a de Rham complex:

$W_0$ →$d_0 = \nabla$ $W_1$ →$d_1 = \nabla\times$ $W_2$ →$d_2 = \nabla\cdot$ $W_3$

This will let us learn flux / circulation / potential balances.

Theorem (Saddle Point) — Abstract Form ★★ The saddle-point problem is: $$\begin{cases}(F, v) - (d_1 u, v) = 0 \\ (\xi, d^* F) = (f, \xi)\end{cases}$$ where $d^*$ is the adjoint — what comes out of IBP.

Example: $(\nabla u, v) = -(u, \nabla \cdot v)$, i.e. $d_0^* = -\nabla\cdot$.
inf-sup stable if $d_k W_k \subseteq W_{k+1}$
§7

Whitney Form Construction

This construction is also known as the Whitney form construction.

Pick $W_0$ as our usual tent functions: $u \in W_0 \Rightarrow u = \sum_i \hat{u}_i\, \phi_i(x)$, with $\phi_i^0(x) = \phi_i(x)$.

Define basis for $W_1$:

$$\psi_{ij}^1(x) = \phi_i \nabla\phi_j - \phi_j \nabla\phi_i$$ $$u \in W_1 \;\Rightarrow\; u = \sum_{i,j} \hat{u}_{ij}\, \psi_{ij}^1(x)$$
Claim: $\nabla W_0 \subseteq W_1$ Proof. Note $\sum_i \phi_i = 1 \Rightarrow \sum_i \nabla\phi_i = 0$. Let $u \in W_0$: $$\nabla u = \sum_i \hat{u}_i \nabla\phi_i = \sum_i \hat{u}_i \left[\nabla\phi_i \left(\sum_j \phi_j\right) - \left(\sum_j \nabla\phi_j\right)\phi_i\right] = \sum_{i,j} \hat{u}_i\, \psi_{ij}^1$$ So $\nabla u \in W_1$. $\quad\square$

Skipping details (for HW): The basis for $W_2$ is:

$$\psi_{ijk}^2 = \phi_k \nabla\phi_i \times \nabla\phi_j + \phi_i \nabla\phi_j \times \nabla\phi_k + \phi_j \nabla\phi_k \times \nabla\phi_i$$

One can show: $u \in W_1 \Rightarrow \nabla\!\times\! u \in W_2$, so the complex property $d_k W_k \subseteq W_{k+1}$ holds throughout.

§8

Hackathon / Homework

For the next hackathon/HW, consider the system with $u \in W_0$, $F \in W_1$:

$$\nabla \cdot F = f, \qquad F = -\nabla u + N[u]$$

The weak form is:

$$(F, v) + (\nabla u, v) - (N[u], v) = 0$$ $$-(F, \nabla\xi) = (f, \xi)$$

If we let $N[u] = \sum_i \mathcal{N}[\hat{u}]_i\, \phi_i^0$, then:

$$\begin{pmatrix} M & G \\ G^T & 0 \end{pmatrix} \begin{pmatrix} \hat{F} \\ \hat{u} \end{pmatrix} + \begin{pmatrix} M\,\mathcal{N}[\hat{u}] \\ 0 \end{pmatrix} = \begin{pmatrix} 0 \\ f \end{pmatrix}$$
Key takeaway The first block is inf-sup stable (solvable, diffusing). The second is the nonlinearity.

Think back to FDM! For a Lipschitz nonlinearity $Au + N[u] = b$ is solvable.

Summary The inf-sup stable discretization using the Whitney complex $W_0, W_1, W_2, W_3$ guarantees that the discrete saddle point problem is well-posed, and the complex property $d_k W_k \subseteq W_{k+1}$ ensures consistency with the continuous de Rham complex.
