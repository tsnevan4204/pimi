Formalizing: $k$-chains and the boundary operator

Definition — $K$-chain
$$C^K = [v_1,\dots,v_K]_{v_i \in V}$$

anti-symmetric with respect to swapping any two entries, i.e.

$$[v_1,v_2] = -[v_2,v_1]$$
Definition — Boundary
$$\partial_k:\ C^{k+1} \to C^k$$
$$\partial_k[v_1,\dots,v_{k+1}] = \sum_{i=1}^{k+1}(-1)^{i-1}\,[n_1,\dots,\widehat{n_i},\dots,n_{k+1}]$$

($\widehat{n_i}$ means: leave this vertex out.)

Example
$$\partial[v_1,v_2] = (-1)^0[v_2] + (-1)^1[v_1] = v_2 - v_1$$
page 2

Geometry → algebra, and the exact sequence property

We can use the boundary operator to turn geometry into algebra. Take a path of two edges:

$$S = [v_1,v_2] + [v_2,v_3]$$ $$\partial S = (v_2-v_1) + (v_3-v_2) = v_3-v_1$$
Theorem — Exact sequence property
$$\partial_{k-1}\circ\partial_k = 0$$
Proof

Let $c \in C^{k+1}$:

$$\partial_{k-1}\partial_k c \;=\; \sum_{i=1}^{k+1} (-1)^{i-1}\,\partial_{k-1}\big[n_1,\dots,\widehat{n_i},\dots,n_{k+1}\big]$$ $$=\ \sum_{i,j}(-1)^{i}(-1)^{j-1}\big[n_1,\dots,\widehat{n_i},\dots,\widehat{n_j},\dots,n_{k+1}\big]$$

↑ this bracket term is antisymmetric under swapping $i \leftrightarrow j$

$$=\ \sum_{i,j} (-1)^{i+j-2}\,\alpha_{ij}$$

↑ the sign prefactor is symmetric in $i,j$; multiplying a symmetric weight against an antisymmetric quantity and summing over all $(i,j)$ kills the sum

$$=\ 0 \qquad \blacksquare$$

A worked example: the triangle, and the coboundary

The 2-chain $C_{ijk}$ and its boundary edges, oriented $i\to j\to k\to i$.

$$C_{ijk} = [v_i,v_j,v_k]$$ $$\partial C_{ijk} = e_{ij}+e_{jk}+e_{ki} = [v_i,v_j]+[v_j,v_k]+[v_k,v_i]$$ $$\partial\partial C_{ijk} = (v_j-v_i)+(v_k-v_j)+(v_i-v_k) = 0$$
To define coboundary (graph div/grad/curl)

Swap the roles of chains and cochains. For $f \in C_k$:

$$\delta_k:\ C_k \to C_{k+1}$$
$$\delta_k f \;=\; \sum_i (-1)^i\, f(v_1,\dots,\widehat{v_i},\dots,v_k)$$

Coboundary examples: grad, curl, and $\mathrm{curl}\circ\mathrm{grad}=0$

Example — grad
$$\delta_0\,\phi_{ij} = \phi_j - \phi_i$$
Example — curl
$$\delta_1\, A_{ijk} = A_{ij} + A_{jk} + A_{ki}$$
$\mathrm{curl}\circ\mathrm{grad} = 0$
$$\delta_1\delta_0\,\phi \;=\; \delta_0\phi_{ij} + \delta_0\phi_{jk} + \delta_0\phi_{ki}$$ $$=\ (\phi_j-\phi_i) + (\phi_k-\phi_j) + (\phi_i-\phi_k)$$ $$=\ 0$$

Codifferentials and the recall table

Definition — Codifferential

Defined as the adjoint, in the $\ell_2$ inner product:

$$(\delta^* f,\, g) = (f,\, \delta g)$$
Recall

$\delta_0 = \mathrm{grad}$,   $\delta_0^* = \mathrm{div}$   |   $\delta_1 = \mathrm{curl}$,   $\delta_1^* = \mathrm{curl}$

$C_0$
$\delta_0$ $\rightleftarrows$ $\delta_0^*$
$C_1$
$\delta_1$ $\rightleftarrows$ $\delta_1^*$
$C_2$

$\delta^*\!\circ\delta^*=0$ and the graph Laplacian

Theorem
$$\delta^*\circ\delta^* = 0$$
Proof — for $f\in C^{k-2},\ g\in C^k$
$$\langle f,\ \delta_k^*\,\delta_{k+1}^*\,g\rangle = \langle \delta_k f,\ \delta_{k+1}^* g\rangle$$ $$=\ \langle \delta_{k+1}\delta_k f,\ g\rangle$$ $$=\ 0$$

Each line just applies the adjoint definition from §6 once; the last line vanishes by the $\delta\circ\delta=0$ identity (the coboundary counterpart of $\partial\circ\partial=0$ — see the $\mathrm{curl}\circ\mathrm{grad}=0$ example above).

We now have a complete graph analog to div / grad / curl. Just as in vector calculus, we can use them to solve Laplacian problems:

Definition — Graph (Hodge) Laplacian
$$\Delta_k = \delta_{k-1}\,\delta_{k-1}^* \;+\; \delta_k^*\,\delta_k$$
$C^{k-1}$
$\delta_{k-1}$ $\rightleftarrows$ $\delta_{k-1}^*$
$C^k$
$\delta_k$ $\rightleftarrows$ $\delta_k^*$
$C^{k+1}$
Example

$k=0$:  $\delta_0^*\delta_0 \;\longrightarrow\; \nabla\cdot\nabla$

$k=1$:  $\delta_1^*\delta_1 + \delta_0\delta_0^* \;\longrightarrow\; -\nabla\times\nabla\times \;+\; \nabla\nabla\cdot$

Application: Hodge decomposition on a graph

In vector calculus

Any field can be split as $F = \nabla\times A + \nabla\phi$ (Hodge / Helmholtz decomposition). How do we find $A,\phi$?

$$\nabla\cdot F = \nabla\cdot\nabla\times A + \nabla^2\phi$$ $$\nabla\times F = \nabla\times\nabla\times A + \nabla\times\nabla\phi$$

→ 2 PDEs to solve

On a graph

Let $f \in C_k$:

$$f = \delta_{k-1}\phi + \delta_k^* A$$ $$\delta_k f = \delta_k\delta_{k-1}\phi + \delta_k\delta_k^* A$$ $$\delta_{k-1}^* f = \delta_{k-1}^*\delta_{k-1}\phi + \delta_{k-1}^*\delta_k^* A$$

From Laplacian to variational problem

We now have the machinery for analysis. Step ① — turn the problem into a variational (weak) one.

Weighted graph Laplacian
$$L^w[u]_i = \sum_j W_{ij}(u_j - u_i) = f_i, \qquad f,u \in C_0,\ \ W_{ij}=W_{ji}$$

Let $v \in C_0$ be a test function. Testing the equation:

$$\langle v,\ L^w[u]\rangle = \langle v, f\rangle$$
$$=\ \sum_{i,j} W_{ij}(u_j-u_i)\,v_i$$ $$=\ \tfrac{1}{2}\sum_{i,j} W_{ij}(u_j-u_i)v_i \;+\; \tfrac{1}{2}\sum_{i,j} W_{ij}(u_i-u_j)v_j$$ $$=\ \tfrac{1}{2}\sum_{i,j} W_{ij}(u_j-u_i)(v_j-v_i)$$

↑ weighted inner product on $C_1$

Bilinear form
$$a(u,v) := \tfrac{1}{2}(\delta v)^{\top} W (\delta v)$$
Lax–Milgram isn't just for fields — it applies here too!

① Coercivity:

$$\|u\|_E^2 = a(u,u) \ge \alpha\langle u,u\rangle = \alpha\|u\|^2$$

② Boundedness:

$$|a(u,v)| \le \gamma\|u\|\|v\|$$

To get $\alpha$ and $\gamma$, we need to understand the eigenvalues.

Rayleigh quotients and eigenanalysis

Let $M = M^{\top}$ with orthogonal eigenpairs $(\lambda_i, v_i)$, $v_i^{\top}v_j = \delta_{ij}$. So:

$$Mv_i = \lambda v_i \quad \text{(definition of eigenpair)}$$ $$v_i^{\top}Mv_i = \lambda v_i^{\top}v_i = \lambda$$
Definition — Rayleigh quotient
$$R(M,x) = \frac{x^{\top}Mx}{x^{\top}x}$$

numerator = energy norm; denominator = the norm we want control over

Expanding $x$ in the eigenbasis, $x = \sum_i \hat g_i v_i$ where $\hat g_i = \langle v_i, x\rangle$:

$$R(M,x) = \frac{\sum_i \lambda_i \hat g_i^2}{\sum_i \hat g_i^2}$$
Min/max of $R$ gives min/max eigenvalues
$$\min_x R(M,x) = \lambda_{\min} = \lambda_{\min}\frac{\sum\hat g_i^2}{\sum\hat g_i^2} \;\le\; \frac{\sum\lambda_i\hat g_i^2}{\sum\hat g_i^2} \;\le\; \lambda_{\max} = \max_x R(M,x)$$

Spectral graph theory: bounding $\lambda_1$

Following Fan Chung, Spectral Graph Theory.

We estimate the smallest nonzero eigenvalue of the graph Laplacian:

$$\lambda_1 = \min_{f \perp \mathbf{1}} \ \frac{\sum_{i,j}(f_i-f_j)^2}{\sum_k f_k^2}$$
  1. 1
    The constant vector has zero eigenvalue: $v_0 = \mathbf{1}$, $\lambda_0 = 0$. By orthogonality, $\langle v_0, f_{\min}\rangle = 0 \;\Rightarrow\; \sum_i f_{\min,i} = 0$.
  2. 2
    Pick $\hat v$ such that $|f(\hat v)| = \max_v |f(v)|$. Because $\sum_i f_i = 0$, we can find a node $\hat u$ such that $f(\hat u)\,f(\hat v) < 0$ (opposite signs).
  3. 3
    Define $D = \max_{i,j\in V} d(v_i,v_j)$, the diameter of the graph. Let $P$ be a path connecting $\hat u$ to $\hat v$ with $m \le D$ edges:
    $$f(\hat v) - f(\hat u) = \sum_{k} f(v_{k+1}) - f(v_k)$$
  4. 4
    By Cauchy–Schwarz:
    $$\big(f(\hat v)-f(\hat u)\big)^2 \;\le\; m\sum_P \big(f(v_{k+1})-f(v_k)\big)^2 \;\le\; D\sum_{i,j}(f_j-f_i)^2$$

    ↑ this right-hand side is exactly the numerator of the Rayleigh quotient.

  5. 5
    Since $f(\hat u)$ and $f(\hat v)$ have opposite signs and $|f(\hat v)| = \max|f(v)|$:
    $$\big(f(\hat v)-f(\hat u)\big)^2 \ge f(\hat v)^2 \;\;\Rightarrow\;\; f(\hat v)^2 \le D\sum_{i,j}(f_j-f_i)^2$$
    For the denominator, since $\hat v$ is the maximizer:
    $$\sum_{k=1}^N f_k^2 \le N f(\hat v)^2$$
Combining steps 4 + 5
$$\lambda_1 = \frac{\sum_{i,j}(f_i-f_j)^2}{\sum_i f_i^2} \;\ge\; \frac{f(\hat v)^2 / D}{N f(\hat v)^2} \;=\; \frac{1}{ND}$$

So the coercivity constant $\alpha$ in Lax–Milgram is $\dfrac{1}{ND}$.
