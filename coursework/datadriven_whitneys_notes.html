Partitions of unity

Definition — Partition of unity (PoU)

A collection of functions $\lambda_i(x): \mathbb{R}^d \to \mathbb{R}$ such that, for all $x$,

$$\lambda_i \ge 0, \qquad \sum_i \lambda_i = 1.$$
Theorem

Partitions of unity are closed under convex combination. (A.k.a. a convex combination of PoUs is a PoU.)

Proof

Let $W \in \mathbb{R}^{N\times M}$ with

$$W_{ij} \ge 0, \qquad \sum_i W_{ij} = 1,$$

and define a new family

$$\psi_i^0(x) = \sum_j W_{ij}\,\lambda_j(x).$$

Is $\{\psi_i^0\}$ a PoU? Clearly $\psi_i^0 \ge 0$, since it's a nonnegative combination of nonnegative $\lambda_j$'s. For the sum:

$$\sum_i \psi_i^0 \;=\; \sum_{i,j} W_{ij}\lambda_j \;=\; \sum_j \Big(\sum_i W_{ij}\Big)\lambda_j \;=\; \sum_j \lambda_j \;=\; 1. \qquad \blacksquare$$
§2 / page 2

Posing the flux problem in weak form

Recall
$$\psi'_{ij} = \psi_i^0\,\frac{d}{dx}\psi_j^0 \;-\; \psi_j^0\,\frac{d}{dx}\psi_i^0$$

and

$$\frac{d}{dx}\,W^0 \subseteq W^1.$$

We search for a flux $F$ and potential $u$ satisfying

$$F = -\nabla u + N[u], \qquad \nabla\cdot F = f,$$

or, in weak form:

$$(F,v) + (\nabla u, v) - (N[u], v) = 0,$$ $$-(F,\nabla z) = (f, z),$$

for every $v \in W^1$, $z \in W^0$.

Notation

Denote this system $\,R(u,F) = 0.$

First contact with graph calculus

Definition — $k$-clique

An ordered $k$-tuple of nodes, anti-symmetric with respect to swap:

$$w \in C_k \;\longrightarrow\; w = (n_i, n_j, \dots)$$
$C_0$
$\partial$ $\longleftarrow$
$C_1$

Example

$$e_{ij} = (n_i, n_j) \in C_1, \qquad (n_j, n_i) = -(n_i,n_j) \in C_1$$ $$\partial(n_i, n_j) = n_j - n_i$$
Definition — Graph functions

Node functions $f_i$, and edge functions $g_{ij} = -g_{ji}$.

Definition — Graph gradient

Acting on node-valued (graph) functions:

$$\delta_0 f_{ij} = f_j - f_i$$

Written as a matrix:

$$(\delta_0)_{e_{ij},\,ab} = \delta_{jb} - \delta_{ia}$$

($\delta_{jb}$ here is the Kronecker delta — the row of $\delta_0$ for edge $e_{ij}$ has a $+1$ in column $j$ and a $-1$ in column $i$, zero elsewhere.)

§4 / page 4

The graph divergence, as the adjoint of the graph gradient

$N$
$\delta_0$ $\longrightarrow$ $\longleftarrow$ $\delta_0^{\,*}$
$E$
Definition — Graph divergence, as adjoint

For all $f, g$:

$$(f,\ \delta_0^{\,*} g) = (\delta_0 f,\ g)$$
Derivation
$$(\delta_0 f, g) = \frac{1}{2}\sum_{i,j} (f_j - f_i)\, g_{ij}$$ $$= \frac{1}{2}\sum_{i,j} f_j\, g_{ij} \;-\; \frac{1}{2}\sum_{i,j} f_i\, g_{ij}$$ $$= \frac{1}{2}\sum_{i,j} f_i\, g_{ji} \;-\; \frac{1}{2}\sum_{i,j} f_i\, g_{ij} \qquad \textit{(reconstructed: relabel } i\leftrightarrow j \textit{ in the first sum)}$$ $$= -\frac{1}{2}\sum_{i,j} f_i\, g_{ij} \;-\; \frac{1}{2}\sum_{i,j} f_i\, g_{ij} \qquad (\text{using } g_{ji} = -g_{ij})$$ $$= \sum_i \Big(\sum_j -g_{ij}\Big) f_i$$ $$= (\delta_0^{\,*}g,\ f)$$

So the graph divergence at node $i$ is the (negative) net outgoing flow over its incident edges:

$$(\delta_0^{\,*} g)_i = -\sum_j g_{ij}.$$
§5 / page 5

From graph gradient back to the continuum

Definition — Graph gradient (restated)
$$\delta_0 f_{ij} = f_j - f_i$$
Etymology

Compare to the continuum line integral of a gradient along the edge $e_{ij}$:

$$\int_{e_{ij}} \nabla \phi \cdot d\ell = \phi_j - \phi_i.$$

The graph gradient is exactly the discrete shadow of this identity.

Note. $\delta_0^{\top} u = \sum_{j \sim i} u_{ij}$ — summing the edge values over all edges incident to node $i$.

As a matrix (as before):

$$(\delta_0)_{e_{ij},\,ab} = \delta_{jb} - \delta_{ia}$$

The discrete bilinear form

In bilinear form, we need to compute the inner product of the derivatives of the basis functions:

$$(u', v) \;=\; \hat v^{\top}\underbrace{\big(\psi^{0\prime},\,\psi^0\big)}_{\displaystyle D_{ij}}\,\hat u$$

Define the mass matrices $M_0$ (on nodes) and $M_1$ (on edges).

Theorem
$$D_{ij} = M_1\,\delta_0$$
Proof (as written; lightly cleaned up)
$$\frac{d}{dx}u = \sum_i \hat u_i\, \frac{d}{dx}\psi_i^0$$ $$= \sum_i \hat u_i \Big(\sum_j \psi_j^0\,\tfrac{d}{dx}\psi_i^0 - \psi_i^0\,\tfrac{d}{dx}\psi_j^0\Big)$$ $$= \sum_{i,j}\hat u_i\,\big(-\psi'_{ij}\big) \;=\; \sum_{i,j} (\hat u_i - \hat u_j)\dots\ \text{grouped into edge form } \psi'_{ij}$$

Pairing against the test function $v$:

$$(v,\ \tfrac{d}{dx}u) \;=\; \hat v^{\top}\Big(\psi^1,\ \textstyle\sum_j \psi'_{ij}\Big)\,\hat u \;=\; \hat v^{\top} M_1\,\delta_0\,\hat u.$$

(This step leans on the same product-rule identity recalled in §2 — $\psi'_{ij}$ is built from $\psi^0_i, \psi^0_j$ exactly so that differentiating the $W^0$ basis produces something expressible in the $W^1$ edge basis, i.e. $\tfrac{d}{dx}W^0 \subseteq W^1$.)

§6 / page 6

Posing a trainable nonlinearity

The question: how do we pose a trainable nonlinear term $N[u]$ so that $(N[u], v)$ stays well-defined?

The naive approach fails

If $N[u]$ were defined directly as the raw output of a neural network — $u \to \boxed{\text{DNN}} \to N[u]$ — then we would not be able to integrate it exactly. That breaks the whole theory built up so far (the exact integration identities are what made the discrete divergence conservative).

Instead

Project the network's output onto the same edge (Whitney-1) basis used everywhere else:

$$N[u] = \sum_{i,j} N[u]_{ij}\,\psi^1_{ij}$$

so that the inner product with any test function reduces to a clean matrix expression:

$$(N[u], v) = \hat v^{\top} M_1\, \widehat{N[u]}$$

The nonlinearity can be any trainable function of $\hat u$ — it just has to be re-expanded in the basis before it's paired against test functions, so the same exact mass-matrix machinery ($M_1$) applies to it as to every other term.

§7 / page 7

Assembling the full system

Putting the flux and balance equations together gives the block system:

$$ \begin{pmatrix} M_1 & M_1\,\delta \\[2pt] -\delta^{\top}M_1 & 0 \end{pmatrix} \begin{pmatrix} \hat F \\ \hat u \end{pmatrix} + \begin{pmatrix} M_N(\hat u) \\ 0 \end{pmatrix} = \begin{pmatrix} 0 \\ f \end{pmatrix} $$

Back-substitution

Eliminating $\hat F$ from the top block row and substituting into the bottom row:

Result
$$-\delta^{\top}M_1\,\delta\,\hat u \;+\; \delta^{\top} M_1\, N(\hat u) \;=\; \text{RHS}$$
Remark

Discrete divergence $\;\Rightarrow\;$ conservation.

Extracting the mass matrices

Let $\hat M_0 = (\phi_i, \phi_j)$ be the Gram matrix of the raw basis $\{\phi_i\}$, and recall the PoU basis is a convex combination of it: $\psi_i^0 = \sum_j W_{ij}\phi_j$. Then:

$$M_0 = (W\phi,\,W\phi) = W^{\top}(\phi,\phi)\,W = W^{\top}\hat M_0\, W$$

and, by a similar (tensorized) formula for the edge mass matrix:

$$M_1 = (W\otimes W)^{\top}\, \hat M_1\, (W\otimes W)$$

When is the system actually solvable?

The data-fitting problem we ultimately want to solve is:

$$\min \;\|u - u_{\text{data}}\|^2 \quad \text{s.t.} \quad R(u,F) = 0$$
Only well-posed if...

...$R$ is itself solvable.

Recall

For a system of the form $Ax + F(x) = b$, we needed

$$\|A^{-1}\| \le C$$

This is the condition covered in the previous lecture.
